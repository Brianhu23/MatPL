# Machine Learning Force Field

## Introduction:

MatPL(PWMLFF) is an open source software under GNU GPL license. It aims at generating force fields with accuracy comparable to Ab Initio Molecular Dynamics (AIMD). It is compatible with AIMD data in either **PWmat** or **VASP** format. 

**Manual**

A complete user manual can be found here: http://doc.lonxun.com/MatPL/
